Lens Schema

Narrative perspective — filters and interprets world content.

Lens

A narrative perspective

PropertyTypeRequiredDescription
perspectiveobject (Perspective)✓Perspective configuration of this lens
lensTypeIRIThe type of this lens, referencing a TypeDefinition
knowledgeobject (Knowledge)Knowledge configuration of this lens
temporalPositionobject (TemporalPosition)Temporal position of the narrator
emotionalobject (Emotional)Emotional coloring of this lens
voiceobject (Voice)Voice characteristics of this lens
reliabilityobject (Reliability)Reliability configuration of this lens
layersobject (LensLayer) (multi)A sub-layer in a composite lens
alternationobject (Alternation)Alternation configuration for composite lens
compositionobject (Composition)Composition configuration for composite lens

Perspective

PropertyTypeRequiredDescription
perspectiveTypeIRI✓Type of perspective, referencing a ValueDefinition
anchorNodeThe node this perspective is anchored to
personIRIGrammatical person, referencing a ValueDefinition (e.g. FIRST, THIRD)

Knowledge

PropertyTypeRequiredDescription
modeIRIKnowledge mode, referencing a ValueDefinition
includeSubconsciousbooleanWhether the narrator has access to subconscious thoughts
includeWrongBeliefsbooleanWhether the narrator includes wrong beliefs

TemporalPosition

PropertyTypeRequiredDescription
temporalTypeIRIType of temporal position, referencing a ValueDefinition
narratingFromstringThe point in time from which the narrator speaks
knowsFutureUntilstringHow far into the future the narrator knows

Emotional

PropertyTypeRequiredDescription
emotionalSourceIRISource of emotional coloring, referencing a ValueDefinition
biasobject (EmotionalBias)Emotional bias toward specific nodes

Voice

PropertyTypeRequiredDescription
vocabularyLevelIRILevel of vocabulary sophistication, referencing a ValueDefinition
sentenceTendencyIRITendency in sentence structure, referencing a ValueDefinition
innerMonologuebooleanWhether inner monologue is used
metaphorDensityIRIDensity of metaphor usage, referencing a ValueDefinition
verbalTicsstring (multi)Characteristic verbal tics of the narrator
languageRenderingobject (LanguageRendering)Language rendering configuration

Reliability

PropertyTypeRequiredDescription
levelIRIReliability level of the narrator, referencing a ValueDefinition
omitsIRI (multi)Event nodes the narrator omits
distortsobject (Distortion) (multi)Distortions applied by the narrator

LensLayer

PropertyTypeRequiredDescription
lensLens✓The lens used in this layer
roleIRI✓Role of this layer in the composite lens
priorityintegerPriority of this layer
triggerJSON objectTrigger condition for this layer

Alternation

PropertyTypeRequiredDescription
alternationModeIRIMode of alternation between sub-lenses, referencing a ValueDefinition
sequenceLensesLens (multi)Ordered sequence of lens references
alternationTriggerIRITrigger for alternation switches, referencing a ValueDefinition

Composition

PropertyTypeRequiredDescription
compositionModeIRIMode of composition between sub-lenses, referencing a ValueDefinition
conflictResolutionobject (ConflictResolution)How conflicts between sub-lenses are resolved

EmotionalBias

PropertyTypeRequiredDescription
towardNode (multi)The node this bias is directed toward
biasStrengthnumberStrength of the emotional bias

LanguageRendering

PropertyTypeRequiredDescription
renderingModeIRIMode of language rendering, referencing a ValueDefinition

Distortion

PropertyTypeRequiredDescription
distortedNodeNodeThe node being distorted
directionIRIDirection of the distortion, referencing a ValueDefinition

ConflictResolution

PropertyTypeRequiredDescription
knowledgeResolutionIRIHow knowledge conflicts are resolved, referencing a ValueDefinition
emotionResolutionIRIHow emotion conflicts are resolved, referencing a ValueDefinition
voiceResolutionIRIHow voice conflicts are resolved, referencing a ValueDefinition
reliabilityResolutionIRIHow reliability conflicts are resolved, referencing a ValueDefinition

Example

{
  "@context": "../../../../context/sac.context.jsonld",
  "type": "Lens",
  "id": "marty-pov",
  "lensType": "SIMPLE",
  "perspective": {
    "perspectiveType": "CHARACTER_BOUND",
    "anchor": "marty",
    "person": "THIRD"
  },
  "knowledge": {
    "mode": "CHARACTER_KNOWLEDGE",
    "includeSubconscious": false,
    "includeWrongBeliefs": true
  },
  "temporalPosition": {
    "temporalType": "IN_MOMENT"
  },
  "emotional": {
    "emotionalSource": "CHARACTER_STATE",
    "bias": {
      "toward": ["george", "lorraine", "doc-brown"],
      "biasStrength": 0.6
    }
  },
  "voice": {
    "vocabularyLevel": "COLLOQUIAL",
    "sentenceTendency": "SHORT_STACCATO",
    "innerMonologue": true,
    "metaphorDensity": "LOW",
    "verbalTics": ["this is heavy", "what the hell"]
  },
  "reliability": {
    "level": "RELIABLE"
  }
}
