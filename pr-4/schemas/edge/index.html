Edge Schema

A relationship between nodes.

Typeobject
RequiredNo
Additional propertiesAny type allowed

Description: A relationship between two nodes that persists over time.

Examples:

id: cave-adjacent-lighthouse
type: SPATIAL
name: adjacent-to
source: hidden-cave
target: lighthouse
directed: false

id: hero-in-dungeon
type: SPATIAL
name: located-in
source: hero
target: dungeon
scope:
  range:
    from:
      type: time
      item: T10
    to:
      type: time
      item: T15

id: hero-has-sword
type: SPATIAL
name: possesses
source: hero
target: magic-sword
scope:
  and:
  - type: frame
    item: alt-timeline
  - range:
      from:
        type: time
        item: T5
      to: null
