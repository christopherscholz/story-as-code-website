Format Schema

Output format definition with structural hierarchy and pacing rules.

Format

Output format definition

PropertyTypeRequiredDescription
namestring✓Display name of an element (shared across many classes)
formatTypeIRI✓The type of this format, referencing a TypeDefinition
structureobject (Structure)✓Structural definition for this format
formatSettingsJSON objectFormat-specific settings

Structure

PropertyTypeRequiredDescription
structureTypeIRI✓Type of structural element, referencing a ValueDefinition
structureConstraintsJSON objectConstraints on this structural level
childrenobject (Structure)Child structural levels

Example

{
  "@context": "../../../../context/sac.context.jsonld",
  "type": "Format",
  "id": "screenplay",
  "name": "Feature Film Screenplay",
  "formatType": "SCREENPLAY",
  "structure": {
    "structureType": "film",
    "structureConstraints": {
      "count": 1,
      "target_runtime": "116 minutes"
    },
    "children": {
      "structureType": "act",
      "structureConstraints": {
        "count": 3
      },
      "children": {
        "structureType": "sequence",
        "structureConstraints": {
          "count": "2-4"
        },
        "children": {
          "structureType": "scene",
          "structureConstraints": {
            "target_runtime": "2-5 minutes"
          }
        }
      }
    }
  },
  "formatSettings": {
    "pacing": {
      "act": {
        "tension_shape": "THREE_ACT"
      },
      "sequence": {
        "tension_shape": "MINI_ARC",
        "end_hook": "USUALLY"
      }
    }
  }
}
