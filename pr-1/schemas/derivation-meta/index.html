Derivation Meta Schema

Validation contract for a compiled output.

Typeobject
RequiredNo
Additional propertiesAny type allowed

Description: Validation contract for a compiled output — links lens, format, and beat back to the world graph state.

Example:

id: chapter-1-output
lens: hero-pov
format: novel-format
beat_position:
  beat: the-call
dependencies:
  nodes:
  - id: hero
    must_exist: true
  edges:
  - id: hero-meets-mentor
    must_exist: true
